# send-a-parcel-api

## Technologies Used
    Java 11
    Spring Boot 2.5.3
    JPA
    MySQL Server 
    Spring Security
    
##  Setup application.properties
    spring.datasource.username=root
    spring.datasource.password=Test123#
    
##  Run Server
    mvn spring-boot:run
    mvn install
